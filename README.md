## 🗺️ Repository Roadmap

This project is divided into three distinct phases, representing the evolution of Natural Language Processing over the last decade.

### 🟢 Phase 1: Basic NLP (The Foundation)
*Focus: Text Preprocessing & Lexical Analysis*
* **Core Concepts:** Tokenization, Stemming vs. Lemmatization, Stop-word removal.
* **Vectorization:** Implementation of **Bag-of-Words (BoW)** and **TF-IDF** (Term Frequency-Inverse Document Frequency).
* **Goal:** Transforming raw, messy strings into structured numerical data that a computer can read.

### 🟡 Phase 2: Intermediate NLP (Semantic Meaning)
*Focus: Statistical Learning & Static Embeddings*
* **Word & Doc Embeddings:** Implementation of **Word2Vec** and **Doc2Vec** to capture semantic similarity.
* **Predictive Modeling:** Training **Naive Bayes, Logistic Regression, and SVMs** for Sentiment Analysis and Text Classification.
* **Information Extraction:** Using **Named Entity Recognition (NER)** to extract structured entities and **Topic Modeling (LDA)** to find hidden themes.
* **Evaluation:** Deep dive into **F1-Scores, Precision-Recall**, and Confusion Matrices.

### 🔴 Phase 3: Advanced NLP (Deep Learning & Transformers)
*Focus: Neural Architectures & Contextual Intelligence*
* **Neural Foundations:** Building **Multilayer Perceptrons** and understanding Backpropagation from scratch.
* **Sequence Modeling:** Overcoming the memory bottleneck using **RNNs and LSTMs**.
* **The Transformer Revolution:** Implementing **Self-Attention** and **Multi-Head Attention** mechanisms.

* **Transfer Learning:** Utilizing SOTA pre-trained models like **BERT** and fine-tuning them for specialized downstream tasks.
